# Version 3.3.3

## Bug Fixes

### Flux2LoraTrainer Indentation Error Fix
- **Fixed**: Indentation error in `decode_vae_latent()` method of `Flux2LoraTrainer`
- **Issue**: Line 924 had incorrect indentation causing `IndentationError: expected an indented block after 'if' statement`
- **Change**: Fixed indentation of `latents = self._unpack_latents(...)` to be properly inside the `if latent_ids is None:` block
- **Impact**: Resolves import error when loading `Flux2LoraTrainer` class

## Technical Details

The `decode_vae_latent()` method had a syntax error where the code after the `if latent_ids is None:` statement was not properly indented. This caused Python to raise an `IndentationError` when trying to import the module.
